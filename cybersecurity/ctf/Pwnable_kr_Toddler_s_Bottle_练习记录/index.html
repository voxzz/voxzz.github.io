fd

查看程序源码，需要使 buf 为 "LETMEWIN" 字符串才可以得到 flag 信息。

在 Linux 系统中，文件描述符使用 int 类型进行表示，其中有三个系统默认值。

  • 0：stdin，标准输入流
  • 1：stdout，标准输出流
  • 2：stderr，标准错误流

当程序中 fd 的值为 0 时，表示 read() 函数将从 stdin 流中读取 32 字节数据到 buf 数组中，此时可以控制 buf 为 "LETMEWIN" 字符串，得到 flag 信息。
